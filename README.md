# School District Analysis

## Overview of the project:

This project analyzes various data regarding the different schools in a school district. The analysis of the school district data comprises of:

1. A summary of the disctrict data.
2. Summary of the statistics of different schools in the district.
3. Math and Reading scores by grades from 9th to 12th.
4. Summary of student performance based on school budget and spending.
5. Summary of student performance based on the size of the school.
6. Summary of student performance based on the type of school.

There are reports that the performance scores for 9th graders at Thomas High School has been altered. This project analyses the data of the district with and without the scores of these 9th graders and makes comparison.

### Purpose:

The analysis of the school district data will enable the board to make better decisions about where to allocate their budget and whether they would need to make any adjustments to their plans for the schools in this district moving forward.
